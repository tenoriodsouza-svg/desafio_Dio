Surgimento da Internet

Voltar ao topo

A internet surgiu nos EUA durante a Guerra Fria (anos 1960) como o projeto militar ARPANET, criado pelo Departamento de Defesa para conectar centros de pesquisa e garantir comunicações descentralizadas e seguras. Evoluiu para uma rede mundial com o protocolo TCP/IP (1983) e a criação da WWW (1990), popularizando-se nos anos 90.

Aqui está um resumo da evolução:

  • 1969 (ARPANET): O marco inicial foi a primeira mensagem enviada entre computadores na Universidade da Califórnia (UCLA) e Stanford. O objetivo era criar uma rede de computadores descentralizada, garantindo que a comunicação sobrevivessse a ataques militares.
  • Década de 1970: Desenvolvimento do e-mail por Ray Tomlinson, introduzindo o símbolo "@", fundamental para a troca de informações.
  • 1983 (TCP/IP): A adoção do protocolo TCP/IP permitiu que diferentes redes se comunicassem, tornando-se o padrão da internet moderna.
  • 1990 (WWW): O cientista britânico Tim Berners-Lee inventou a World Wide Web (WWW) no laboratório CERN, criando o primeiro navegador, sites e a linguagem HTML, tornando a rede acessível.
  • Anos 1990-2000 (Popularização): Lançamento do navegador Mosaic (1993), surgimento de buscadores como Google (1998) e o boom da banda larga facilitaram o acesso comercial e ao público geral.

A rede saiu do ambiente militar/acadêmico e se expandiu mundialmente, transformando-se em uma infraestrutura global baseada em cabos submarinos de fibra óptica.

Como funciona o HTML

Voltar ao topo

O HTML (HyperText Markup Language) foi criado pelo físico britânico Tim Berners-Lee em 1991, no centro de pesquisa CERN, na Suíça, com o objetivo de permitir que pesquisadores compartilhassem documentos e informações de forma simples através da World Wide Web (WWW). Baseado em padrões como SGML, o HTML evoluiu de uma forma estrutural para a atual versão, o HTML5.

Principais marcos na origem do HTML:

  • 1991: Tim Berners-Lee cria o HTML e o primeiro navegador.
  • 1995: Aparece o HTML 3.2, com suporte a tabelas e imagens.
  • 1997: O HTML 4.0 é lançado, trazendo melhorias na apresentação e acessibilidade.
  • 2008: O HTML5 é introduzido, com novas tags e funcionalidades para criar aplicações web mais ricas.

O que são Tags

Voltar ao topo

Tags HTML são blocos de construção básicos da linguagem HTML (HyperText Markup Language) que definem a estrutura, o significado e a organização do conteúdo de uma página web, como títulos, parágrafos, imagens e links. Elas são delimitadas por sinais de menor e maior (ex: ), agindo como instruções para os navegadores.

  • Estrutura: As tags geralmente vêm em pares, com uma tag de abertura (ex: <>) e uma tag de fechamento (ex: ), embora algumas sejam auto-fecháveis (ex: ).
  • Função: Elas indicam ao navegador como exibir o conteúdo, como formatar texto, inserir imagens ou criar links.
  • Exemplos comuns:
    • <h1> a <h6>: Títulos de diferentes níveis.
    • <p>: Define um parágrafo.
    • <a>: Cria um link para outra página ou recurso.
    • <img>: Insere uma imagem na página.

    • As tags HTML são os blocos de construção fundamentais da web, permitindo que desenvolvedores criem páginas interativas e visualmente atraentes.

    Entender os diferentes tipos de tags HTML e como elas são usadas é importante. Ao fazer isso, você estará mais equipado para criar e modificar páginas da web de forma eficaz.

    Embora as tags HTML ofereçam muitas vantagens, é importante estar ciente de suas limitações e desafios. Ao combiná-las com outras tecnologias web, é possível criar experiências digitais envolventes e acessíveis para todos os usuários.
